## 💡 Навички та компетенції

### 🔹 Vanilla JavaScript (ES6+)

Стрілкові функції, шаблонні рядки, деструктуризація, тернарний оператор, область видимості `const`/`let`, `String.prototype.trim`, методи масивів (`map`, `forEach`, `join`).

### 🔹 Володіння DOM API

- **Вибірка:** `querySelector`, `querySelectorAll`, `children`
- **Читання:** `textContent`, `value`, `event.target`
- **Запис:** `insertAdjacentHTML`, інлайн-`style`
- **Продуктивність:** одноразова вставка в DOM замість поелементного `append` у циклі

### 🔹 Програмування на основі подій

- `addEventListener` для подій `input`, `submit`, `click`
- Запобігання дефолтній поведінці через `event.preventDefault()`
- Читання динамічного вводу через `event.target`

### 🔹 Робота з формами

- Доступ до полів через `form.elements` (замість окремих селекторів)
- Власна логіка валідації без атрибута `required`
- Структурний збір даних в об'єкт
- Скидання стану форми через `form.reset()`

### 🔹 Якість коду та робочий процес

- **Модульна архітектура** — кожне завдання ізольоване у власній трійці HTML/CSS/JS
- **Prettier** для уніфікованого форматування
- **Git** з описовими, атомарними комітами
- **GitHub Pages** для безперервного деплою

## 🧩 Розбір функціональності

### 1. Інспектор категорій

Обходить вкладений ненумерований список та виводить його структуру в консоль — загальну кількість категорій, назву кожної категорії та кількість елементів у ній. Демонструє навігацію по DOM-дереву через `children` та ітерацію колекції з `forEach`.

```
Number of categories: 3
Category: Animals      → Elements: 4
Category: Products     → Elements: 3
Category: Technologies → Elements: 5
```

---

### 2. Динамічна галерея зображень

Будує повноцінну галерею з масиву даних JavaScript. Демонструє **підхід з огляду на продуктивність**: уся розмітка формується в пам'яті та вставляється в DOM **однією операцією**, а не поелементним додаванням у циклі.

```js
const markup = images
  .map(
    ({ url, alt }) =>
      `<li class="gallery-item"><img src="${url}" alt="${alt}" /></li>`,
  )
  .join("");

galleryEl.insertAdjacentHTML("beforeend", markup);
```

---

### 3. Живий інпут привітання

Відображає введення користувача на сторінці у реальному часі. Реалізує захисний UX-паттерн: коли поле порожнє або містить лише пробіли, вивід коректно показує `"Anonymous"` замість залишатися порожнім.

---

### 4. Форма логіна

Форма з власною валідацією. Обробник `submit` перехоплює стандартну відправку, перевіряє обидва поля та або сповіщає користувача про порожні поля, або збирає очищені від пробілів значення в об'єкт даних. Демонструє роботу з `form.elements` — ідіоматичний спосіб читати кілька полів одночасно.

```js
{
  email: "user@example.com",
  password: "secret123"
}
```

---

### 5. Генератор випадкового кольору

За кожним кліком на кнопку генерує випадковий HEX-колір, застосовує його як фон `<body>` через інлайн-стиль та показує значення користувачеві. Демонструє маніпуляції з інлайн-стилями та чистий патерн утилітних функцій.

```js
function getRandomHexColor() {
  return `#${Math.floor(Math.random() * 16777215)
    .toString(16)
    .padStart(6, 0)}`;
}
```

## 🚀 Локальний запуск

Жодного build-пайплайна, менеджера пакетів чи залежностей — просто відкрий.

