## 💡 Навыки и компетенции

### 🔹 Vanilla JavaScript (ES6+)

Стрелочные функции, шаблонные строки, деструктуризация, тернарный оператор, область видимости `const`/`let`, `String.prototype.trim`, методы массивов (`map`, `forEach`, `join`).

### 🔹 Владение DOM API

- **Выборка:** `querySelector`, `querySelectorAll`, `children`
- **Чтение:** `textContent`, `value`, `event.target`
- **Запись:** `insertAdjacentHTML`, инлайн-`style`
- **Производительность:** одноразовая вставка в DOM вместо поэлементного `append` в цикле

### 🔹 Программирование на основе событий

- `addEventListener` для событий `input`, `submit`, `click`
- Предотвращение дефолтного поведения через `event.preventDefault()`
- Чтение динамического ввода через `event.target`

### 🔹 Работа с формами

- Доступ к полям через `form.elements` (вместо отдельных селекторов)
- Собственная логика валидации без атрибута `required`
- Структурный сбор данных в объект
- Сброс состояния формы через `form.reset()`

### 🔹 Качество кода и рабочий процесс

- **Модульная архитектура** — каждое задание изолировано в своей тройке HTML/CSS/JS
- **Prettier** для единого форматирования
- **Git** с описательными, атомарными коммитами
- **GitHub Pages** для непрерывного деплоя

## 🧩 Разбор функциональности

### 1. Инспектор категорий

Обходит вложенный ненумерованный список и выводит его структуру в консоль — общее количество категорий, название каждой категории и количество элементов в ней. Демонстрирует навигацию по DOM-дереву через `children` и итерацию коллекции с `forEach`.

```
Number of categories: 3
Category: Animals      → Elements: 4
Category: Products     → Elements: 3
Category: Technologies → Elements: 5
```

---

### 2. Динамическая галерея изображений

Строит полноценную галерею из массива данных JavaScript. Демонстрирует **подход с учётом производительности**: вся разметка формируется в памяти и вставляется в DOM **одной операцией**, а не поэлементным добавлением в цикле.

```js
const markup = images
  .map(
    ({ url, alt }) =>
      `<li class="gallery-item"><img src="${url}" alt="${alt}" /></li>`,
  )
  .join("");

galleryEl.insertAdjacentHTML("beforeend", markup);
```

---

### 3. Живой инпут приветствия

Отображает ввод пользователя на странице в реальном времени. Реализует защитный UX-паттерн: когда поле пустое или содержит только пробелы, вывод корректно показывает `"Anonymous"` вместо того чтобы оставаться пустым.

---

### 4. Форма логина

Форма с собственной валидацией. Обработчик `submit` перехватывает стандартную отправку, проверяет оба поля и либо уведомляет пользователя о пустых полях, либо собирает очищенные от пробелов значения в объект данных. Демонстрирует работу с `form.elements` — идиоматический способ читать несколько полей одновременно.

```js
{
  email: "user@example.com",
  password: "secret123"
}
```

---

### 5. Генератор случайного цвета

По каждому клику на кнопку генерирует случайный HEX-цвет, применяет его как фон `<body>` через инлайн-стиль и показывает значение пользователю. Демонстрирует манипуляции с инлайн-стилями и чистый паттерн утилитных функций.

```js
function getRandomHexColor() {
  return `#${Math.floor(Math.random() * 16777215)
    .toString(16)
    .padStart(6, 0)}`;
}
```

## 🚀 Локальный запуск

Никакого build-пайплайна, менеджера пакетов или зависимостей — просто открой.

